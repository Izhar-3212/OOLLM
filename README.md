# TinyLlama Chatbot Project

This folder contains a starter project structure for building and experimenting with a TinyLlama-based chatbot.

## Structure
- data/raw: source documents
- data/processed: cleaned data
- data/training: final training format
- models/base: base TinyLlama model
- models/fine-tuned: your fine-tuned model
- models/gguf: quantized model for inference
- scripts: training and inference pipeline

## Next steps
1. Add your source documents to `data/raw/`.
2. Review `config.yaml` and update paths/model settings.
3. Run the preparation and training scripts in order.
